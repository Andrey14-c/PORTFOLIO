Experiencia

PRIMERA EXPERIENCIA

Tengo experiencia básica en la elaboración y comprensión de Diagramas de Flujo de Datos (DFD). Estoy familiarizado con los niveles de diagramas y con los principales componentes, como procesos, almacenes de datos, entidades externas y flujos de datos. He utilizado DFDs para representar de forma clara y estructurada el flujo de información en sistemas sencillos, lo que me ha permitido mejorar la organización y análisis de requerimientos en proyectos de desarrollo de software.

SEGUNDA EXPERIENCIA

Tengo experiencia básica en el uso de PSeInt, una herramienta educativa diseñada para apoyar el aprendizaje de la lógica de programación. He utilizado PSeInt para desarrollar algoritmos sencillos que resuelven problemas comunes, aplicando estructuras básicas como secuencias, condicionales, ciclos y manejo de variables. Esta experiencia me ha permitido fortalecer mi lógica de programación y prepararme para el uso de lenguajes de programación más avanzados.

[translate:Mentoría Open Source]

Tengo experiencia básica en HTML y CSS, lo que me ha permitido crear y estructurar páginas web sencillas. Conozco las etiquetas fundamentales de HTML para organizar contenido (como encabezados, párrafos, listas, enlaces e imágenes), así como las principales propiedades de CSS para aplicar estilos, modificar el diseño visual y trabajar con aspectos como colores, fuentes, márgenes y posicionamiento. Esta base me ha ayudado a entender el funcionamiento de la web y sentar las bases para continuar aprendiendo desarrollo DE software
